# Type-pairing database

Curated, distinctive pairings: a display face (headings, brand), a body face (text, UI), and a mono (numbers, code, labels). All are free and self-hostable. Never Inter, Roboto, or Arial as the primary display. Pick by mood, then commit.

Sources: **Fontshare** (Indian Type Foundry: Clash Display, Satoshi, General Sans, Cabinet Grotesk, Boska, Tanker, Switzer, Sentient) · **Google Fonts** (Instrument Serif, Fraunces, Bricolage Grotesque, Hanken Grotesk, Manrope, Playfair Display, Newsreader, Spectral, Syne, Unbounded, DM Sans, Space Grotesk, Big Shoulders) · **Vercel** (Geist, Geist Mono). Mono: JetBrains Mono, IBM Plex Mono, Space Mono, Geist Mono.

Format: `Display + Body + Mono — mood. Why.`

## App, product, premium
- **Clash Display + Satoshi + JetBrains Mono** — refined app, confident. Why: the MVP pairing, geometric authority + clean body.
- **General Sans + Switzer + Geist Mono** — modern product, neutral-plus. Why: characterful grotesk, warm body.
- **Cabinet Grotesk + Satoshi + IBM Plex Mono** — precise, engineered. Why: strong display, the Bitcoin pairing.
- **Geist + Geist + Geist Mono** — techy, Vercel-clean (vary it, do not default here). Why: cohesive single-family, mono labels.

## Editorial, publishing, luxury
- **Instrument Serif + Hanken Grotesk + IBM Plex Mono** — elegant editorial. Why: high-contrast serif headline, clean body.
- **Fraunces + Hanken Grotesk + JetBrains Mono** — Swiss-editorial, warm. Why: optical serif, the Vigie pairing.
- **Playfair Display + Manrope + IBM Plex Mono** — classic luxury. Why: didone gravitas, geometric body.
- **Newsreader + Switzer + Space Mono** — journal, reading-first. Why: text serif, calm sans, quirky mono captions.
- **Spectral + General Sans + IBM Plex Mono** — literary, considered. Why: a serif with personality + neutral sans.
- **Boska + Satoshi + JetBrains Mono** — fashion, high-contrast display. Why: dramatic display serif, clean body.
- **Sentient + Switzer + Geist Mono** — calm premium serif. Why: soft humanist serif, modern body.

- **Unbounded + Hanken Grotesk + JetBrains Mono** — playful-bold, rounded. Why: rounded display energy.

## Playful, friendly, consumer
- **Unbounded + Manrope + IBM Plex Mono** — app for humans, round. Why: friendly rounded display.
- **Bricolage Grotesque + DM Sans + JetBrains Mono** — modern consumer. Why: warm character without childishness.
- **Tanker + Switzer + Space Mono** — bold display, fun. Why: chunky display, clean body.

## Institutional, trust, finance, legal
- **Fraunces + Hanken Grotesk + IBM Plex Mono** — credible-editorial. Why: serif gravitas, readable body.
- **Newsreader + Manrope + IBM Plex Mono** — institutional, calm. Why: text serif + neutral sans.
- **Inter (as body only) + a grotesk display + IBM Plex Mono** — the conservative client exception. Why: when a client demands the safe road, use Inter for body only, never as the display. The LeCoffre case (Inter/Poppins/Montserrat) is this conservative end, chosen deliberately, not a default.

## Mono-forward, dev, data
- **Space Grotesk + Switzer + JetBrains Mono** — dev product (vary, Space Grotesk is overused). Why: geometric display, mono numerals.
- **General Sans + Satoshi + Geist Mono** — data dashboard, neutral. Why: clean, numbers in mono tabular.
- **IBM Plex Sans + IBM Plex Serif + IBM Plex Mono** — coherent IBM family, technical. Why: one super-family, three voices.

## Rules when picking
- Pair a display with a body: two voices, one conversation. Self-host as woff2, `font-display: swap`.
- Numbers (prices, balances, stats, addresses) go in the mono, tabular.
- Never default to the same pairing twice across clients. Never Space Grotesk as the converged choice; if you reach for it, vary it.
- Letter-spacing -0.01 to -0.02em on large headings; body 16px, line-height 1.5.
